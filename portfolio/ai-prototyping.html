Background

The Challenge: The Static Design Bottleneck

Traditional design workflows led to significant "Design-to-Dev" overhead because static frames could not simulate "Human-in-the-loop" feedback systems. This created critical blockers:

  • Technical Gap: Designers lacked the environment to test non-deterministic AI behaviors like real-time streaming and confidence signaling.
  • Scaling Friction: Prototyping was fragmented across teams, making it difficult for engineers to re-use design code.
  • Resource Constraints: High-end AI tools were required, but a strategic business case was needed to justify the investment.

Leadership: Building the Infrastructure

I managed a team of four designers and served as the strategic lead for our technical onboarding environment.

  • The Strategy: I architected a pipeline that allows designers with zero technical background to leverage the Evisort Design System (EDS) in a live code environment.
  • The Implementation: I developed a custom GitHub repo and template that standardized the root folder structure, ensuring high code reusability for engineers and seamless onboarding for designers.
  • Code Connect: I collaborated with Brian and Eva to lead the strategy for Code Connect, ensuring all AI prototypes utilized live EDS core components.

Product Strategy & Implementation

The "AI-Ready" Standard: Machine-Readable Layers

I defined strict "pre-processing" requirements to ensure our design intent was accurately captured by AI coding agents. I mandated 100% Auto-layout and systematic naming to allow tools like Cursor and Claude to ingest designs without errors.

AI Prototyping Workflow: This pipeline bridges static designs and production code via Figma MCP and Cursor for rapid iteration and GitHub hosting.

Evisort AI Design System (EADS)

I codified reusable patterns for non-deterministic AI states, including streaming text, feedback loops, and reasoning cards. This ensured AI agents utilized live production components through Code Connect.

Code Connect for AI Prototyping: Technical architecture linking Figma components to the EDS codebase through Storybook to ensure AI agents utilize live production components.

The Force Multiplier: Global Enablement

I shifted the culture from static mockups to functional prototyping through continuous education and mentorship.

  • Systemic Scaling: I hosted 3+ intensive workshops and demos, successfully ramping up the entire design team and Evisort PMs.
  • Mentorship: I introduced and onboarded interns and their respective teams—including Katie's Reporting & Analytics Design squad—by providing specialized AI resources.

Iteration workflow for team education: An environment enabling real-time React JS generation from Figma frames to test non-deterministic AI behaviors and interactive feedback loops.

Executive Summary of Impact

  • High-Value Adoption: The infrastructure is now the global standard for AI features across the Workday/Evisort ecosystem.
  • Operational Efficiency: Reduced R&D waste by establishing a "Human-in-the-loop" standard that validates AI concepts through functional code before engineering handoff.
  • Platform Maturity: Successfully bridged the gap between static design and production-ready code, enabling the product org to iterate at unprecedented velocity.
